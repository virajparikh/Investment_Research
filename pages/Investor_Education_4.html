Adding Pages

1.

Find this web site's files on your computer

They will be in your Dropbox folder under Apps/Backlift/

2.

Create a new .html file in the /pages folder

The easiest way is to copy the /pages/blank_page_2.html file and rename it.

The new page will automatically be detected and added to the navigation menu. All numbers and underscores will be stripped out of the filename in order to create the link text.

If you need menu items to be ordered a particular way, put a number in the filename. For example: services_1.html, contact_2.html, about_3.html.

3.

Make some changes!

Try making some changes to your new file. Your page will refresh automatically as you work.

Next, for a peak under the hood, check out the /config.yml file.
